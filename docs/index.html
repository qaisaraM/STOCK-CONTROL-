core features
What It Does
🔐
Multi-Role Authentication
Face recognition login for PE members via MTCNN + LBPH. Password-based access for PIC and Approvers. Guest signup with approval workflow.
📦
Real-Time Stock Tracking
Live stock in/out recording. Safety stock alerts with automated low-stock email notifications to PIC and Approver 1.
✅
2-Tier Approval System
Structured order approval through Approver 1 → Approver 2. Toggle approve/reject per item. Auto PDF report + email summary on final approval.
📊
Analytics Dashboard
Interactive Matplotlib charts: drill orders by period, cost trends, top-10 usage, type breakdown — with hover tooltips and popup zoom.
🔄
Multi-User File Sync
QFileSystemWatcher + 2s polling keeps all open sessions in sync across the network share — no server required.
📧
Outlook Email Automation
Automated notifications via win32com for order submissions, approval decisions, delivery reminders, and guest account approvals.

workflow
Order Approval Flow
01
PIC identifies low stock & submits order
OrderManagement page flags items below safety stock. PIC sets quantities, checks items, and submits — generating an Order ID and emailing Approver 1.
02
Approver 1 reviews & approves per line item
Toggle approve/reject on each item individually. Approved items advance to Approver 2 with an automated email notification.
03
Approver 2 gives final decision
Final authority. On submission, auto-generates a PDF summary and emails all stakeholders (PIC, Approver 1, Approver 2).
04
PIC records delivery & updates stock
Order History page shows approved orders awaiting delivery. Recording a delivery auto-increments current stock and logs the transaction.
